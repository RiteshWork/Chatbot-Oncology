================================================================================
HOW IT WORKS
================================================================================

1. Bot sends first message (introduction)
2. You respond naturally
3. System classifies your emotion (calm/anxious/neutral)
4. AI generates warm, supportive response
5. State automatically transitions to next step
6. Repeat through all 6 steps
7. Session ends at closing affirmation

Total time: 20-30 minutes

================================================================================
TROUBLESHOOTING
================================================================================

Error: "connection refused on port 5555"
   → PostgreSQL not running. Start it first.

Error: "no pq wrapper available"
   → Run: pip install --upgrade psycopg[binary] --break-system-packages

Error: "Process 'calm_v1' not found"
   → Run: python scripts/create_sample_data.py

Error: "module not found"
   → Make sure you're in the relaxbot directory
   → Check requirements.txt is installed
